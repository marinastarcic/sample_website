Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
div division group elements to format them with CSS
header header include a container for introductory content
ol Orderd list rank-order
ul unorderd list bullet points

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>strong text</strong> strong text
a anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code Show text as code <code>code text</code> code text
span span group inline elements <span>Call me Ishmael.</span> Call me Ishmael.

    Document Tags

    Tag Name Purpose
    html html tells the browser that this is an HTML document
    head head head element can contain title for the document, scripts, styles, meta info. etc.
    body body body tag defines the document's body
    title title defines a title in the browser toolbar
    meta metadata metadata is info about data

    Divs and Spans

    Tags that have little to no impact on page appearance, but help organise the page into logical units.

    Tag Name Purpose
    header header not to be confused with head; abstract semantic tag used to label part of the page, has no immediate effect on page's appearance
    div division block element; set the divisions apart
    span span inline element; like div but for inline use
    blockquote blockquote for quoting blocks of text
